# Foneettinen nimikone

Nimikoneen linkki: [esiivola.github.io/nimikone](https://esiivola.github.io/nimikone)

**Foneettinen nimikone** auttaa löytämään etunimiä, jotka sopivat yhteen valitun sukunimen kanssa. Työkalu hyödyntää foneettista analyysiä sopivien nimien suosittelemiseen. Suosittelun lisäksi nimiä voi suodattaa niiden ominaisuuksien perusteella (esim. suosio eri maissa tai suosio eri vuosikymmenillä).

Suodattamisen lisäksi sivu näyttää kaikista suomessa käytössä olevista etunimistä tilastoa niiden suosiosta 1900-luvun alusta lukien, sekä antaa hyvän yleiskuvan nimenkantajien ikäprofiilista, sekä nimen foneettisista piirteistä. 

Nimikone on luotu, sillä suomalaisia nimiä on liikaa, ja netissä saatavilla olevat vaihtoehtoiset nimensuosittelusivut eivät toimi. Nimikone sopii tuoreille vanhemmille, jotka etsivät lapselleen täydellistä nimeä ja haluavat suodattaa varmasti huonot vaihtoehdot helposti pois.

Sivun käyttö vaatii salasanan. Voit pyytää salasanaa minulta.
